¿Qué es CSS?

Es algo para diseñar y cambiar la presentación de un sitioweb.

¿Qué es una referencia por palabra clave?

Cuando quieres personalizar una parte especifica como por ejemplo body, nav etc.

¿Qué es una referencia por id?

ID se puede usar para identificar un solo elemento

¿Qué es una referencia por clase

Cuando quieres que ciertas zonas se modifiquen y a todas las que quieres que se modifiquen les pones esa palabra clave para identificarla,se puede usar para agrupar más de uno.
